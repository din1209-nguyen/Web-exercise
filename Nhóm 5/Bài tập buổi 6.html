BUỔI 06 — BÀI TẬP THỰC HÀNH
Xây dựng server + form đăng ký người dùng

ĐỀ BÀI

Node.js + Express · Middleware · Routing · HTML + Fetch API

YÊU CẦU

Viết Express server có middleware pipeline và 2 route. Tạo thêm file index.html với 2 form gọi API bằng fetch() và hiển thị kết quả ngay trên trang — không reload.

CẤU TRÚC PROJECT

index.js — public/index.html — .env — .gitignore

BACKEND — index.js

MW
Logger — ghi [time] METHOD /path ra console cho mọi request
MW
checkAge — age < 18 hoặc không có → trả 400 + message lỗi, ngược lại next()
GET
/api/info?name=&age=Gắn checkAge. Trả JSON: name, age, lời chào mừng
POST
/api/register — body: name, age, emailValidate không bỏ trống. Trả lại thông tin + id tự tăng
STATIC
express.static('public') — serve file HTML cho trình duyệt

FRONTEND — public/index.html

FORM 1
Input name + age → gọi GET /api/info bằng fetch → hiện kết quả hoặc lỗi ngay trên trang
FORM 2
Input name + age + email → gọi POST /api/register bằng fetch → hiện thông tin đăng ký thành công

GỢI Ý

  • 1
    Middleware phải đặt trước route — Express chạy theo thứ tự khai báo
  • 2
    Route GET gắn middleware riêng: app.get('/api/info', checkAge, handler)
  • 3
    Bắt response: .then(res => res.json()) rồi đẩy vào innerHTML
  • 4
    Nhớ event.preventDefault() để trang không reload khi submit
